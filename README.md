# PythonStudy

## Setting up

**Anaconda**
  - Download Anaconda
  - Go to "Environments" -> "base(root)" -> Open Terminal
  - Now can run Python in cmd: 
    - ``Python --version``
    - ``python -m ensurepip --default-pip`` or ``python -m pip install --upgrade pip setuptools wheel`` 
    
    - ``conda install pip``
    - ``pip install --upgrade setuptools``
  - Install packages
    - ``pip install cvxpy``
